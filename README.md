# ThinkingViT Project Page

This repository contains the GitHub Pages website for:

**ThinkingViT: Matryoshka Thinking Vision Transformer for Elastic Inference**
CVPR 2026 Main Track Poster

Authors: Ali Hojjat, Janek Haberer, Soren Pirk, Olaf Landsiedel

## Links

- Project page: https://ds-kiel.github.io/ThinkingViT-project-page/
- Paper: https://arxiv.org/abs/2507.10800
- Code: https://github.com/ds-kiel/ThinkingViT
- Model weights: https://zenodo.org/records/20207165

## Local Preview

From this directory:

```bash
python3 -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

## Deploy

This repo is intended to be deployed with GitHub Pages from:

```text
Branch: main
Folder: / root
```

After editing the page:

```bash
git add .
git commit -m "Update ThinkingViT project page"
git push origin main
```
